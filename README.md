# Netflix-Data-Analysis
Netflix Data Analysis is an exploratory project analyzing Netflix movies and TV shows. It uncovers trends in release years, genres, countries, and content types, with clear visualizations. The project demonstrates data cleaning, visualization, and analytical skills, making it GitHub-ready and portfolio-worthy.

## Features
- Analyze Netflix dataset for insights
- Visualize data using graphs and charts
- Identify top genres, top-rated shows, and release trends
- Understand Netflix content distribution over time

## Installation

1. Clone the repository:
   git clone https://github.com/Ajay92146/Netflix-Data-Analysis.git
2. Navigate to the project folder:
   cd Netflix-Data-Analysis
3. Install required Python packages:
   pip install -r requirements.txt
4. Run the analysis:
   python netflix_analysis.py

## Dataset
The project uses the Netflix dataset containing shows and movies metadata like title, genre, release year, and ratings.
